== Description ==
Form Vibes is a submission manager plugin that you can use to save your submissions from Contact Form 7 and other form building plugins like Elementor Form, Beaver Form, etc., into the database. It also provides an analytics report to track how different contact forms are performing over a period of time!

### Form Vibes Features

1. [Capture Form Submissions](https://docs.formvibes.com/article/96-using-form-vibes-getting-started?utm_source=wporg&utm_medium=fv&utm_campaign=readme)- It automatically captures and saves the contact form submissions in the database.
2. [Viewing Submissions](https://docs.formvibes.com/article/77-viewing-form-submissions?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - View your captured submissions in a tabular format.
3. [Export Submissions](https://docs.formvibes.com/article/80-exporting-form-submissions-into-csv-file?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - You can export the submissions in a separate CSV file.
4. [Delete Submissions](https://docs.formvibes.com/article/78-deleting-form-entries?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - Delete unimportant form entries from the submissions table.
5. [Analyses Form Submissions](https://docs.formvibes.com/article/82-analytics?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - Analyses the contact form performance graphically.
6. [Dashboard View](hhttps://docs.formvibes.com/article/81-dashboard-view?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - View quick stats and analytics on your WordPress Admin Dashboard.
7. [Date Range Filters](https://docs.formvibes.com/article/83-using-date-range-filter?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - Filter submissions monthly, weekly, daily, or for a specific date.
8. [Auto-Refresh](https://docs.formvibes.com/article/98-form-vibes-settings-page?utm_source=wporg&utm_medium=fv&utm_campaign=readme) - The submissions table is automatically updated.

2. Activate the plugin through the 'Plugins' screen in WordPress

To know more details, checkout the [Getting Started Guide](https://docs.formvibes.com/article/76-installing-form-vibes/?utm_source=wporg&utm_medium=fv&utm_campaign=readme)

 == Frequently Asked Questions ==

= Does it need any configuration? =

No, it is a plug 'n' play plugin. Just install the plugin, and it will start capturing and displaying reports from supported form plugins.
You can visit the settings page to see some basic config options.

= Will this plugin work if Contact Form 7 is not installed? =

No, this plugin work only when Contact Form 7 is installed & activated.

= How can I display contact form entries on the front end? =

You can display entries on the front end with a simple shortcode available in settings.

= How can I view all contact form entries? =

You can view form entries from the main WordPress dashboard just by clicking on Form Vibes.

= Can I export the form entries into CSV? =
Yes, it's pretty simple; just click on the Quick Export option on the Form Vibes toolbar to download the submissions.

= How can I report security bugs? =

You can report security bugs through the Patchstack Vulnerability Disclosure Program. The Patchstack team help validate, triage and handle any security vulnerabilities. [Report a security vulnerability.](https://patchstack.com/database/vdp/form-vibes)
